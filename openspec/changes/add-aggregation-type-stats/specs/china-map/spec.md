## MODIFIED Requirements

### Requirement: Markers 标记点
系统 SHALL 支持在地图上显示充电桩标记点，并根据地图层级自动聚合，聚合时展示各类型充电桩数量。

#### Scenario: 聚合点类型统计
- **WHEN** 地图处于聚合视图（全国/省/市级）
- **THEN** 系统统计该区域内汽车充电桩和电瓶车充电桩的数量
- **AND** 聚合点标签显示类型图标和数量（如 "🚗5 🛵3"）

#### Scenario: 聚合点 Tooltip 类型明细
- **WHEN** 用户悬停在聚合点上
- **THEN** Tooltip 显示区域名称和各类型充电桩数量
- **AND** 格式为 "区域名\n汽车充电桩: X\n电瓶车充电桩: Y"

#### Scenario: 区级视图散点
- **WHEN** 地图处于区级视图
- **THEN** 系统显示该区内所有充电桩的散点标记
- **AND** 汽车充电桩和电瓶车充电桩使用不同样式
